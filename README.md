# AirOnePage React  

A modern, accessible, mobile-first, and SEO-friendly single-page website built with **React, SCSS, Material UI, and SwiperJS**.  

🔗 **Live Demo:** [AirOnePage React](https://tpanasiuk.github.io/aironepage_react/)  

---

## 🚀 Features  

✅ Fully responsive & mobile-first design  
✅ Accessibility-focused (ARIA, keyboard navigation)  
✅ Optimized for **performance & SEO**  
✅ Modular & reusable React components  
✅ SCSS with **BEM naming conventions**  
✅ Animated statistics counter  
✅ SwiperJS-powered carousel  
✅ Material UI for dialogs & UI components  
✅ Deployed on GitHub Pages
